Manual VICO OS

Bienvenido a VICO POCKET OS. Esta es una aplicación Local-First diseñada para la productividad y el análisis.

1. Links Pocket (Share Target): El corazón de la app. Desde cualquier otra aplicación (Chrome, Twitter...), usa "Compartir" y selecciona VICO POCKET para guardar el enlace directamente con etiquetas.

2. Discovery Pro: Lee feeds RSS. Usa los botones de cada noticia para: Guardar a Links, Compartir externamente, Añadir a Favoritos o enviar a Análisis ROI.

3. Sistema VICO Core: En Perfil, personaliza sonidos, fuentes e idioma. Activa el Modo Dev para funciones experimentales.
